# tanishk-portfolio

Personal portfolio site. Live at [tanishk-portfolio-opal.vercel.app](https://tanishk-portfolio-opal.vercel.app/).

Built with Next.js and Tailwind CSS. Deployed on Vercel.

## What's here

- **Case study:** Strategic Synthesis Engine. An enterprise document intelligence system designed for a regulated biotech client. 12-week deployment plan, full PRD, and parallelized roadmap across six workstreams.
- **Architecture notes:** Separate repo documenting design decisions from building a private RAG platform with dual-model inference, grounded generation, and a verification layer. [notes-on-private-rag-systems](https://github.com/tanishkthorat/notes-on-private-rag-systems)
